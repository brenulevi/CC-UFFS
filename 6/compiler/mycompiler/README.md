## My Compiler

- Tokens <br>
    - Variable declaration
        > [type] [name] => [value];

        - =>
        - ;
        - [type] = int || float || char || bool
        - [name]
        - [value]

    - Statement
        > [variable] [operator] [variable]
        > [variable] [operator] [constant]

        - [variable]
        - [operator] = + || - || = || * || / || != || < || > || >= || <= || ==
        - [constant]

    - Conditional Structure
        > if ([condition1]) {
            [action1]
        } elseif ([condition2]) {
            [action2]
        } else { [action3] }

        - if
        - elseif
        - else
        - [condition]
        - [action]

    - For/while
        > for ([variable declaration]; [condition]; [increment]) {
            [action]
        }

        - for
        - ;
        - [condition]
        - [increment]
        - [action]

        > while ([condition]) do
        {
            [action]
        }

        - while
        - do
        - [condition]
        - [action]

        > do { [action] } while([condition]);
